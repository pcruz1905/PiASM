
# PiASM: Raspberry Pi Assembly & C Machine Project

This repository contains code and resources for making a Raspberry Pi machine work using a combination of C and Assembly language. The project is focused on low-level programming, direct hardware interaction, and educational exercises that demonstrate how to control and manage a Raspberry Pi at the bare-metal or systems level.


## Repository Structure

- **src/main/**: Core C and Assembly source files, machine manager, and supporting resources for the Raspberry Pi.
- **test/**: Test files and data for validating machine manager and assembly routines.
- **USAC01 - USAC11/**: Individual project folders, each with specific exercises or assignments, typically with their own `main.c`, `asm.h`, and `Makefile`.


## Getting Started

1. **Requirements:**
   - Raspberry Pi (or compatible emulator)
   - GCC or compatible C compiler
   - GNU assembler (as)
   - Make
   - (Optional) An IDE or text editor with assembly/C support

2. **Building a Project:**
   - Navigate to the desired project folder (e.g., `src/main/` or `USAC01/`).
   - Run `make` to build the project.
   - Run the resulting executable as instructed in the project folder or deploy to your Raspberry Pi.

3. **Testing:**
   - Some projects include test files in the `test/` directory. Use `make` or follow instructions in the test folder to run tests.
